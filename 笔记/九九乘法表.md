#### shell脚本

九九乘法表

```shell
#!/bin/bash
<<mmm
九九乘法表
mmm

for x in {1..9}
do
    for y in $(seq 1 $x)
    do
      echo -ne "$y*$x=$[$y*$x]\t"
    done
    echo
done
```

说明：echo -n 表示不换行输出，-e表示支持反斜线字符转换

#### while循环

```shell
# 写一个计算1+100累积的脚本
#!/bin/bash
i=1
sum=0
while [ $i -lt 100 ]
do
  sum=$[$sum+$i]
  i=$[$i+1]
done
echo $sum
```

##### while标准输入

将/etc/passwd添加行号并重定向生成file1.txt

```shell
cat -n /etc/passwd > file1.txt
```

查看生成后的file1文件

每0.3秒一次循环列出文件内容

```shell
for h in $(cat file1.txt)
do
echo $h
sleep 0.3
done
```

编写脚本

```shell
#!/bin/bash
cat file1.txt | while read nicaicai
do
        echo $nicaicai
        sleep 0.3
done
```
